# knu-web-web-3

## Running the lab

### 1. Install the dependencies
```
npm install
```

In the `interface` directory:
```
npm install
```
### 2. Deploy the contract
```
truffle migrate
```
### 3. Copy all files from the `build/contracts` to `interface/contracts`

### 4. Now you can start the interface
```
node interface/main.js
```
